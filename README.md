# Phoenix Releases

This repository contains an archive of firmware releases. It is preferred that users use the dropdown interfaces in Phoenix Tuner X to minimize any confusion.

## Releases

Releases are announced through various mediums, but users can subscribe to notifications for a release by "watching" this repository or through our [RSS feed](https://api.ctr-electronics.com/rss/rss.xml).

## Documentation

User documentation, API documentation, code examples, RSS feeds, direct tooling downloads and more can be found at our [documentation landing page](https://docs.ctr-electronics.com/).

## Issues

For a timely response to your issue, please reach out to us directly through our [contact us](https://store.ctr-electronics.com/pages/contact) page.
